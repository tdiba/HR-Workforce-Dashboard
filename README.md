### Project Overview:
This project presents an interactive HR Workforce Dashboard built in Power BI to analyze employee demographics, attrition patterns, job roles, performance metrics, and workforce distribution.
The objective was to transform raw HR data into a structured, interactive business intelligence dashboard that supports strategic workforce planning and HR decision-making.

### Project Objectives
- Analyze employee distribution by department and role
- Monitor attrition trends
- Evaluate workforce demographics
- Examine performance and tenure patterns
- Enable interactive filtering for HR insights
- Create executive-ready visual reporting


### Dataset Description
Files Used:
- HRM database PowerBI.xlsx – Primary HR dataset
- GEO DATA.xlsx – Geographic mapping dataset
- HR Workforce Dashboard in PowerBI.pbix – Power BI dashboard file


### Data Preparation and Transformation
Before dashboard creation, the following steps were performed in Power BI (Power Query):

1. Data Cleaning
   - Removed unnecessary columns
   - Standardized text fields
   - Handled missing values
   - Ensured consistent data types
  
2. Data Modeling
   - Established relationships between HR and GEO datasets
   - Created structured data model
   - Optimized fields for reporting
  
3. Calculated Columns & Measures (DAX)
   - Attrition Rate
   - Average Age
   - Headcount Metrics
   - Tenure Calculations
   - Performance Aggregations
  

### Dashboard Features
The HR Workforce Dashboard includes:
- Total Headcount Overview
- Attrition Analysis
- Department-Level Breakdown
- Geographic Distribution (Map Visualization)
- Gender & Age Distribution
- Performance & Tenure Insights
- Interactive slicers for dynamic filtering

The dashboard enables HR managers and executives to explore workforce trends in real time



### Skills and Technologies Used
- Power BI
- Power Query
- DAX
- Microsoft Excel



### Key Insights Identified
- Attrition varies across departments and roles
- Certain job functions show higher turnover risk
- Workforce distribution differs significantly by location
- Tenure and performance show measurable patterns
- Demographic breakdown reveals diversity distribution trends



### Business Relevance
This dashboard supports:
- Strategic workforce planning
- Attrition risk identification
- HR policy development
- Diversity & inclusion analysis
- Executive reporting
- Resource allocation decisions


### Limitations
- Dataset is static and not connected to a live HR system
- No predictive attrition modeling included
- Performance metrics limited to available variables
